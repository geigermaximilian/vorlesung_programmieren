##Hello World - Besprechung:

public class D01_HelloWorld{
    public static void main(String[] args){
        String text; //Deklaration

        text="Hello World" //Initialisierung

        System.out.println(text); //Ausgabe
    }
}

- Klassennamen müssen mit einem großen Buchstaben beginnen:
    --> public class D01_HelloWorld
    --> void: weil kein Rückgabewert

##Notizen

- Datenobjekt: Container, worin etwas gespeichert wird.
- Datentypen: - primitive Datentypen: (hart in Java implemintierte Datentypen - kleine Anfangsbuchstaben)
                - *int*, short, byte, long _(ganze Zahlen)_
                - *double*, float _(Gleitkommazahlen)_
                - boolean _(Wahrheitswert)_
                - char _(einzelnes Zeichen)_

                ! man kann Datentypen nicht abfragen, die noch nicht initialisiert wurden.
                ! man kann in einen Datentyp nur genau einen Wert speichern.

- float float1 = 0.1F; //F nötig, da sonst double
- long long1 = 5_000_000_000L; //L nötig, sonst int
